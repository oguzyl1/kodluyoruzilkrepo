# 3 ve 4'e Tam Bölünen Sayıların Ortalaması

Bu program, kullanıcıdan alınan bir sayıya kadar olan sayılar içerisinde 3 ve 4'e tam bölünebilen sayıların aritmetik ortalamasını hesaplayan bir Java uygulamasıdır.

## Ödev İçeriği

Program şu mantıkla çalışır:
- Kullanıcıdan bir üst sınır sayısı alınır.
- 0'dan bu sayıya kadar bir `for` döngüsü çalıştırılır.
- Hem 3'e hem de 4'e kalansız bölünen (yani 12'nin katı olan) sayılar tespit edilir.
- Bu sayıların toplamı, bulunan sayı adedine bölünerek ortalama ekrana yazdırılır.

## Nasıl Çalıştırılır?

Klasör içindeki `Main.java` dosyasını derleyip çalıştırabilirsiniz:

```bash
javac Main.java
java Main
```

### Örnek Senaryo
- Girilen Sayı: 24
- 3 ve 4'e bölünenler: 12, 24
- **Sonuç:** (12 + 24) / 2 = 18.0
