# Basamak Sayılarının Toplamı

Bu program, kullanıcının girdiği bir tamsayının basamaklarındaki rakamların toplamını hesaplayan bir Java uygulamasıdır.

## Ödev İçeriği
Girilen bir sayının rakamlarını elde etmek için mod (%) ve bölme (/) operatörleri kullanılmıştır.

**Örnek:**
- **Sayı**: 1643
- **Hesaplama**: 1 + 6 + 4 + 3 = 14
- **Çıktı**: 14

## Nasıl Çalıştırılır?

Klasör içindeki `Main.java` dosyasını derleyip çalıştırabilirsiniz:

```bash
javac Main.java
java Main
```
