# Palindrom Sayı Bulma

Bu program, kullanıcının girdiği bir tamsayının "palindrom sayı" olup olmadığını kontrol eden bir Java uygulamasıdır.

## Palindrom Sayı Nedir?
Palindromik sayı, iki taraftan okunduğu zaman okunuş yönüyle aynı olan sayılardır.

**Örnekler:**
- **101** (Tersten okunuşu da 101)
- **4004** (Tersten okunuşu da 4004)
- **9889** (Tersten okunuşu da 9889)

## Nasıl Çalıştırılır?

Klasör içindeki `Main.java` dosyasını derleyip çalıştırabilirsiniz:

```bash
javac Main.java
java Main
```

### Örnek Senaryo
- **Girdi**: 363 -> **Çıktı**: 363 bir palindrom sayıdır.
- **Girdi**: 123 -> **Çıktı**: 123 bir palindrom sayı değildir.
