# Dizideki Elemanların Harmonik Ortalaması

Bu program, bir tamsayı dizisindeki elemanların harmonik ortalamasını hesaplayan bir Java uygulamasıdır.

## Harmonik Ortalama Nedir?
Harmonik ortalama, bir veri kümesindeki değerlerin gözlem sayısının, bu değerlerin terslerinin toplamına bölünmesiyle elde edilir.

**Formül:**
`Harmonik Ortalama = n / (1/x1 + 1/x2 + ... + 1/xn)`

## Ödev İçeriği
Sınıf örneğinde yapılan aritmetik ortalama yerine, harmonik seri formülü kullanılarak daha hassas bir ortalama hesabı yapılmıştır.

## Nasıl Çalıştırılır?

Klasör içindeki `Main.java` dosyasını derleyip çalıştırabilirsiniz:

```bash
javac Main.java
java Main
```
