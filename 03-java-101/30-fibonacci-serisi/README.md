# Fibonacci Serisi Bulan Program

Bu program, kullanıcının girdiği eleman sayısına göre Fibonacci dizisini hesaplayan ve ekrana yazdıran bir Java uygulamasıdır.

## Fibonacci Serisi Nedir?
Fibonacci serisi, 0'dan başlayan ve her sayının kendisinden önceki iki sayının toplamıyla oluştuğu bir sayı dizisidir.

**Örnek Mantık:**
- 0 + 1 = 1
- 1 + 1 = 2
- 1 + 2 = 3
- 2 + 3 = 5
- 3 + 5 = 8
- ...

## Nasıl Çalıştırılır?

Klasör içindeki `Main.java` dosyasını derleyip çalıştırabilirsiniz:

```bash
javac Main.java
java Main
```
