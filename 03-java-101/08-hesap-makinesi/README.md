# Hesap Makinesi

Bu program, kullanıcıdan alınan iki sayı üzerinde seçilen aritmetik işlemi (Toplama, Çıkarma, Çarpma, Bölme) geçekleştiren bir Java uygulamasıdır.

## Ödev İçeriği

Programda `if-else` yerine **`switch-case`** yapısı kullanılmıştır.

**Desteklenen İşlemler:**
1. Toplama
2. Çıkarma
3. Çarpma
4. Bölme (0'a bölünme kontrolü dahil)

## Nasıl Çalıştırılır?

Klasör içindeki `Main.java` dosyasını derleyip çalıştırabilirsiniz:

```bash
javac Main.java
java Main
```

### Örnek Senaryo
- Sayı 1: 10
- Sayı 2: 5
- Seçim: 3 (Çarpma)
- **Sonuç:** Çarpma : 50
