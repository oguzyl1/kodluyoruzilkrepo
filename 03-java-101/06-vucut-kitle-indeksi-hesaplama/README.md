# Vücut Kitle İndeksi Hesaplama

Bu program, kullanıcıdan boy ve kilo değerlerini alarak "Vücut Kitle İndeksi" (VKİ) değerini hesaplayan bir Java uygulamasıdır.

## Ödev İçeriği

Program şu formülü kullanarak hesaplama yapar:
- **Formül** : Kilo (kg) / (Boy(m) * Boy(m))

## Nasıl Çalıştırılır?

Klasör içindeki `Main.java` dosyasını derleyip çalıştırabilirsiniz:

```bash
javac Main.java
java Main
```

### Örnek Girdi/Çıktı
- Boy: 1,72
- Kilo: 105
- **Çıktı:** Vücut Kitle İndeksiniz : 35.49215792320173
