# Harmonik Seri Hesaplama

Bu program, kullanıcının girdiği bir `n` sayısına kadar olan harmonik serinin toplamını hesaplayan bir Java uygulamasıdır.

## Harmonik Seri Formülü
**result = 1 + 1/2 + 1/3 + 1/4 + ... + 1/n**

## Ödev İçeriği
Döngüler kullanılarak her adımda `1.0 / i` değeri toplama eklenmiştir. Tam sonuç elde edebilmek için `int` yerine `double` veri tipi kullanılmıştır.

## Nasıl Çalıştırılır?

Klasör içindeki `Main.java` dosyasını derleyip çalıştırabilirsiniz:

```bash
javac Main.java
java Main
```

### Örnek Senaryo
- **Girdi**: 5
- **Hesaplama**: 1 + 1/2 + 1/3 + 1/4 + 1/5 = 2.283333333333333
- **Sonuç**: 2.283333333333333
