# Çin Zodyağı Hesaplama

Bu program, kullanıcının doğum yılını alarak 12 hayvanlı Çin Zodyağı burcunu hesaplayan bir Java uygulamasıdır.

## Ödev İçeriği

Çin Zodyağı, doğum yılının 12 ile bölümünden kalan sayıya göre hesaplanır.

**Hesaplama Tablosu:**
- Mod 0: Maymun
- Mod 1: Horoz
- Mod 2: Köpek
- Mod 3: Domuz
- Mod 4: Fare
- Mod 5: Öküz
- Mod 6: Kaplan
- Mod 7: Tavşan
- Mod 8: Ejderha
- Mod 9: Yılan
- Mod 10: At
- Mod 11: Koyun

## Nasıl Çalıştırılır?

Klasör içindeki `Main.java` dosyasını derleyip çalıştırabilirsiniz:

```bash
javac Main.java
java Main
```

### Örnek Senaryo
- Doğum Yılı: 1990
- Hesaplama: 1990 % 12 = 10
- **Sonuç:** Çin Zodyağı Burcunuz : At
