# Armstrong Sayı Bulma

Bu program, kullanıcının girdiği bir sayının "Armstrong Sayı" olup olmadığını kontrol eden bir Java uygulamasıdır.

## Armstrong Sayı Nedir?
N haneli bir sayının basamaklarının n’inci üstlerinin toplamı, sayının kendisine eşitse, böyle sayılara Armstrong sayı denir.

**Örnekler:**
- **407**: (4^3) + (0^3) + (7^3) = 64 + 0 + 343 = 407 (Armstrong sayıdır)
- **1634**: (1^4) + (6^4) + (3^4) + (4^4) = 1 + 1296 + 81 + 256 = 1634 (Armstrong sayıdır)

## Nasıl Çalıştırılır?

Klasör içindeki `Main.java` dosyasını derleyip çalıştırabilirsiniz:

```bash
javac Main.java
java Main
```
