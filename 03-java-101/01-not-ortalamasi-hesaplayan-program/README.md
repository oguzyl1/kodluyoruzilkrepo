# Not Ortalaması Hesaplayan Program

Bu program, kullanıcıdan farklı derslerin notlarını alan ve bu notların ortalamasını hesaplayıp ekrana yazdıran basit bir Java uygulamasıdır.

## Ödev İçeriği

Program şu derslerin notlarını kullanıcıdan ister:
- Matematik
- Fizik
- Kimya
- Türkçe
- Tarih
- Müzik

**Ek Özellik:** Ortalamayı hesapladıktan sonra, eğer ortalama 60'tan büyükse ekrana "Sınıfı Geçti", küçükse "Sınıfta Kaldı" yazdırır.

## Nasıl Çalıştırılır?

Klasör içindeki `Main.java` dosyasını derleyip çalıştırabilirsiniz:

```bash
javac Main.java
java Main
```
